# Au Feu Dormant - Console RP V2

Prototype vierge pense comme une application RP pratique pour joueurs et MJ.

## Idee centrale

- Le joueur peut modifier sa fiche narrative.
- Le joueur peut repartir uniquement les points disponibles.
- Le joueur peut accepter des quetes.
- Le MJ peut modifier la progression : XP, niveaux, points, competences, talents, factions, bonus et quetes.
- Tout est sauvegarde localement dans le navigateur avec `localStorage`.

## Lancement local

Depuis ce dossier :

```powershell
python -m http.server 4190
```

Puis ouvrir :

```text
http://127.0.0.1:4190/
```

## Important

Ceci est un prototype local. Les mots de passe, droits et donnees ne sont pas encore securises pour un vrai site public.
